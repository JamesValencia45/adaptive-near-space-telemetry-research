# Adaptive Near-Space Telemetry Research

Investigation into telemetry reliability, antenna systems, and adaptive communication architectures for high-altitude and near-space platforms.

<br>

## Research Goals

This repository documents an ongoing research project exploring communication reliability in dynamic environments.

Areas of investigation include:

- Telemetry architectures
- Antenna systems
- Adaptive communication strategies
- Atmospheric effects on communication links
- High-altitude communication systems
- Weather balloon validation platforms
